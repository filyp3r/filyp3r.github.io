Reutilização de Código

Para que serve isso?

Muitas vezes, precisamos utilizar um código que está em outra página do projeto. Se não utilizarmos frameworks, como o React por exemplo, a nossa alternativa se torna ctrl + c e ctrl + v. O problema é que isso se torna algo frustrante no futuro, quando a página precisar de manutenção. Por isso esse artigo.

Como reutilizar um código sem copiar e colar?

Eu uso uma técnica que eu descobri brincando com o javascript, é um simples uso da função document.write().

Inicialmente se é criado um arquivo .js. Dentro do arquivo deve conter o document.write(), sendo como parâmetro, o código entre acentos graves. Esse acento grave é utilizado para pular linhas durante o código, porquê tanto com aspas simples, quanto com aspas duplas, pular linhas não funciona.

Logo após é só chamar o código no mesmo lugar onde o código deveria estar e pronto!

Observação: Se por acaso houver links que redirecione para páginas dentro do projeto, utilize caminhos relativos.

Como o código deve parecer:

Arquivo .js

document.write(`
    Código aqui
`);

Arquivo .html

<!DOCTYPE html>
<html lang="pt-br">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Nome do Documento</title>
</head>
<body>
    <script src="NOME DO ARQUIVO .js"><script>
</body>
</html>
